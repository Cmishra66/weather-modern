 Weather App

A simple React app to check the weather in any city. Built with Vite, Tailwind CSS, and OpenWeatherMap API.


Tech Used

- React + Vite  
- Tailwind CSS  
- Axios  
- OpenWeatherMap API


 How to Run

1. Clone this repo  
2. Run `npm install`  
3. Create a `.env` file with your API key:
